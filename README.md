# lagos
Lagos State Performance Dashboard

A modern, fast, and responsive analytics dashboard built with React + Vite.
This dashboard provides an intuitive interface for visualizing Lagos State performance metrics across key sectors such as Infrastructure, Economy, Education, Health, Environment, and more.


Features

Lightning-fast frontend powered by Vite

Dynamic sliders & data visualization for comparing local government performance

Modern UI with clean layout and Lagos branding

OpenGraph and SEO-ready metadata for sharing

Fully responsive (mobile, tablet, desktop)

Reusable React components

API-ready architecture for backend integration


| Category           | Tools                                           |
| ------------------ | ----------------------------------------------- |
| Frontend Framework | React + Vite                                    |
| Language           | TypeScript                                      |
| Styling            | TailwindCSS |
| State Management   | React Hooks             |
| Deployment         | Netlify        


Installation
1. Clone the repository
git clone https://github.com/kcblaq/lagos.git
cd lagos

2. Install dependencies
npm install

3. Start server
   npm run dev

Live: https://lagos-lgas.netlify.app/
